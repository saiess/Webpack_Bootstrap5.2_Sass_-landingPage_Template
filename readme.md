# Webpack App Landing Page

Includes the following features...

- Babel Loader
- Sass Loader
- HTMLWebpackPlugin
- BundleAnalyzerPlugin
- WebpackDevServer
- Asset Resource Loader
- Source Maps
- Caching
- Bootstrap 5.2
- postcss-loader

## Usage

(Port 3000)
```
  yarn dev
```

## Build for production

```
yarn build
```
